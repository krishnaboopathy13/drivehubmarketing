FAQ
What are the 13 interview statuses?▼
Registered, Checked In, Screening, In Progress, Round 1 Interview, Round 2 Interview, HR Round, Offer Waiting, Selected, On Hold, Rejected, No Show, Blacklisted. Each with color-coded pills and status history tracking.
How does the queue system work?▼
When security checks in a candidate, they get a queue number (#1, #2, #3...) based on entry time. First come, first serve. Queue position is visible to admin, recruiter, security, and the candidate themselves via Track Status.
Can recruiters update candidate status?▼
Yes. Recruiters log in with their phone number and see their assigned candidates. Each candidate card has 8 action buttons (Screening, Round 1, Round 2, HR, Offer, Selected, Hold, Reject) plus a notes field for interview feedback.
What does security desk do?▼
Security logs in with a 4-digit PIN. They scan/type token numbers. Only 2 buttons: Check In (records entry time + queue number) and No Show. Security doesn't manage interview status — that's the recruiter's job.
What's the pricing?▼
India: Starter ₹2,999/mo, Growth ₹9,999/mo, Pro ₹29,999/mo, Enterprise custom. Global: $499/mo Pro, $1,999/mo Enterprise. All plans start with a 7-day free trial, no credit card needed.
Does it work offline?▼
Yes. All data stored in IndexedDB. Candidates register, security checks in, recruiters update status — all without internet. Syncs automatically when reconnected.
